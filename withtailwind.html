📘 What is API, HTTP & URL?

  • Protocol: Protocol ကတော့ Client နဲ့ Server တို့ အပြန်အလှန်ဆက်သွယ်oို့ အသုံးပြုကြမယ့် ကြားခံ ဆက်သွယ်ရေးနည်းပညာ ပါ။
  • HTTP: The web’s language — used to request data. HTTP လို့ အတိုကောက် ခေါ်ကြတဲ့ Hypertext Transfer Protocol ကိုသုံးကြတာပါ။ Client နဲ့ Server တို့က HTTP ကိုအသုံးပြုပြီး ဆက်သွယ်အလုပ်လုပ်ကြ
  • API: A way for apps to talk to each other and get data. API ဆိုတာ Application Program Interface ရဲ့ အတိုကောက် ဖြစ်ပြီးတော့၊ Service များကို ရယူအသုံးပြုရန် သတ်မှတ်ထားသောနည်းလမ်း ဖြစ်ပါတယ်။
  • URL: URL = Uniform Resource Locator. It's just a web address you use to request something from the internet.

💡 Why async/await Is Best

  • ✅ Cleaner and easier to read than chained .then()
  • ✅ Handles errors with simple try/catch
  • ✅ Looks like normal step-by-step code
  • ✅ Best for complex flows or if you call many APIs in a row
